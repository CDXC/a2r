# a2r

**a2r** is a command line tool made to convert an arabic number to a roman numeral.

## Installation:

```bash
$ git clone https://github.com/CDXC/a2r.git
$ cd a2r
$ make 
$ sudo make install
```
then you can safely delete the **a2r** directory:

```bash
$ cd .. && rm -r a2r
```

## Removal:
```bash
$ sudo make clean
```
## Usage:

```bash
$ a2r [number]
```
